# 🌸 Iris Flower Classification 

An interactive Machine Learning web application built with **Streamlit** that predicts the species of an Iris flower based on user-input features (sepal length, sepal width, petal length, petal width).

---
## 📌 Problem Statement
The Iris dataset is a classic benchmark in machine learning. This project demonstrates how supervised learning can classify flower species using simple morphological features.

---

## 🚀 Live Demo
👉 [Open the App](https://iris-flower-classification-odhxijwkzf2a2t8febeaxh.streamlit.app/)  


---
## ✨ Features
- Interactive sliders to input sepal and petal dimensions
- Real‑time predictions of Iris species
- Random Forest model trained on the Iris dataset
- Deployed on Streamlit Cloud for easy access

---
